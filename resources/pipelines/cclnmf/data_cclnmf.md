### CCL-NMF: Coupled Cross-section Longitudinal Nonnegative Matrix Factorization

#### Input:

- T1 scan (one or multiple, required)

#### Output:

- CCL-NMF component loading coefficients (.csv)

#### Example:
```
📁 my_project
│
📥 Input
├── 📁 t1

│   └── 📄 scan2_T1.nii.gz
│
📤 Output
├── 📁  CCLNMF_csv
│   └── 📄 CCLNMF.csv
```
