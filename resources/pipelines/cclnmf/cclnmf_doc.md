## input:
- T1 scan (one or multiple, required)

## output:
- SurrealGAN scores (single csv file with SurrealGAN score for each scan)

   
## example:
📁 my_project
│
📥 Input
├── 📁 t1
│   ├── 📄 scan1_T1.nii.gz
│   └── 📄 scan2_T1.nii.gz
│
📤 Output
├── 📁 surreal-gan
│   └── 📄 surreal-gan.csv
