### RAVENS: Abnormality Mapping of Regional Volumes

#### Source:

https://github.com/CBICA/NiChart_RAVENS

#### Description

RAVENS uses T1 scans and DLMUSE regional brain volumes to generate abnormality maps calculated from the NiChart reference data

#### Input

- T1-weighted scan (Nifti)
- Demographics CSV containing Age and Sex (CSV)

#### Output

- Abnormality mapping (Nifti)
